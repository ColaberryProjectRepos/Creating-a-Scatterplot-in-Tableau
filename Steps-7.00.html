Using an Action Filter

From the source report, Sales/Profits by Customer, the user can filter to only see customers by region or view all regions at once.


When a customer is selected from the Sales/Profit by Customer worksheet, the user is taken to a secondary worksheet, Customer Details which reflects the how the customer is performing relative to other customers across all regions, the sub-categories that the customer has purchased, and the profit for the sales.
