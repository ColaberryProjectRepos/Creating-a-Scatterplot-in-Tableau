Creating a Action Filter

In order to allow the user to gain even more details about the sales and profits, a drill-through report will be created.  When the user clicks on a specific customerâ€™s name, the report will drill- through to a more detailed report giving key information about that customer including how the customers sales and profits measure up against the average customer, the product category that they purchased from and the dollar amount of profits they generated.  

Start by making a duplicate copy of the current worksheet. To do so, click the Duplicate icon at the top of the worksheet (9th from the left). This will create and exact copy of the current worksheet. Click the tab at the bottom to rename the new worksheet to avoid confusion. For this example, the new work sheet will be named Customer Details. 

Now, we will set up the drill through capabilities. At the top of the worksheet, click the worksheet tab, scroll down to actions and select. A new Actions pop-up will appear. Click on Add Actions and select Filter. 

From the Add Filter Actions pop-up, make the following revisions. First, give the filter a meaningful name. Under Source Sheets, select the worksheet that the user will start on. Change the Run Action On option to Select. Next, select the Target Sheet. This is the worksheet that the original worksheet will drill-through to display. After these changes have been made, select Ok. 

Additional detail information will be added to this report but first, unnecessary information will be removed. Because this report is a drill down based on the customer name, the Name details and label pills can be removed. This can be done by right clicking on the pill under the marks shelf and clicking remove. You can also click the pill and drag it to the white space beneath the shelf to remove. 


Now, additional detailed information can be added to the report. Drag Color Analysis, Product Sub-Category, and Profit to Label.
