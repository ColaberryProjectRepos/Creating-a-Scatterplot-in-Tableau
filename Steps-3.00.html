Applying a Filter 3

Applying a filter will allow the data to be examined in further detail. In this instance, the data will be filtered by region. By adding a filter to the Region dimension, the user will be able to select which region for which they would like to view data.

Click and drag the Region Dimension to the Filters shelf.  In the pop-up, select the regions to appear in the filter. In this instance, all regions are selected. Click apply and okay for changes to take place. 


To add user functionality, right click on the Region pill and select show filter. 



A filter will appear to the right side of the screen. The user will be able to select individual regions to use in the visualization by selecting or de-selecting the associated check boxes.
