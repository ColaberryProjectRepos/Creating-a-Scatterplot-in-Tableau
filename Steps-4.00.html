Adding a Reference Line

A reference line can be used to divide the scatterplot plane into user defined quadrants. Reference lines can also be helpful when creating a visualization that reflects key targets such as sales goals or budget requirements. In this instance, the reference line will indicate the average profit and average sales for all the customers across all regions.

Right click on the x-axis and select Add Reference Line. 

In the pop-up, the line will divide the plain based on the average sales for all customers in all regions. Label has been set to none. Click ok. Repeat the same steps for the Y-axis. This will divide the data field by the average profits for all customers in all regions.
