Creating a Calculated Field

Calculated fields in Tableau are fields that are created; they don’t already exist in the data source. Calculated fields are created using formulas that are often based on other fields. In this instance, a calculated field will be created that bring color coded categories to the visualization based on the customer’s sales/profits in relation to the average of all customers in all regions.

In the Measures area, right click in the white space to get the pop-up menu. Select Create calculated field. 

In the pop-up, give the calculated field the name,  Color Analysis and create the applicable formula. (Formula can be found by clicking the code text link) Click apply. In this instance, the field will be created to correspond with the reference lines. 

Once the Color Analysis calculated field has been created, drag this pill to the color shelf.  The colors will change based on the quadrant the mark falls in. 

By clicking Color on the Marks shelf, the color palette can be changed so that the colors correspond to the sales/profit position. 

For this example, the Traffic Light pallet to reflect high sales/profits as Green and low sales/profits as red.
